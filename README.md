## 🌟 Inspiration Behind the Project

During a visit to a local pizza café, I was genuinely impressed by the quality of the pizzas, the welcoming atmosphere, and the overall dining experience. The attention to detail in both the food and the environment inspired me to recreate that experience digitally.

**Grotto** is a modern frontend web application that captures the essence of a premium pizza restaurant. The goal was to build a visually engaging and responsive website that reflects the warmth, elegance, and appeal of a real-world café.

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone <your-repository-url>
```

### Install Dependencies

```bash
bun install
```

### Run the Development Server

```bash
bun run dev
```

### Build for Production

```bash
bun run build
```

### Preview the Production Build

```bash
bun run preview
```

### Run Tests

```bash
bun run test
```
