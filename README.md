# Grotto Frontend Code

A Vite + React + TypeScript frontend scaffold with Tailwind CSS and shadcn/ui-inspired components.

## Project overview

- React 18 app built with Vite.
- Tailwind CSS for styling.
- Radix UI and shadcn-style component primitives.
- React Router for page routing.
- Vitest for testing.

## Getting started

Install dependencies:

```bash
bun install
```

Run the dev server:

```bash
bun run dev
```

Build for production:

```bash
bun run build
```

Preview the production build:

```bash
bun run preview
```

Run tests:

```bash
bun run test
```

## Project structure

- `src/main.tsx` — application entry point.
- `src/App.tsx` — root app component.
- `src/pages/` — page-level route components.
- `src/components/` — shared UI components and shadcn-style primitives.
- `src/assets/` — static assets.
